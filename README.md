# React — Ciclo de Vida dos Componentes

Projeto de estudo sobre o ciclo de vida dos componentes em React, criado com Vite.

## Sobre

Este repositório contém exemplos e exercícios que demonstram os diferentes estágios do ciclo de vida de componentes React (montagem, atualização e desmontagem), usando componentes funcionais e hooks.

## Tecnologias

- React
- Vite
- JavaScript (ESModules)

## Requisitos

- Node.js (v16+ recomendado)

## Como executar

1. Clonar o repositório

2. Instalar dependências:

   npm install

3. Iniciar o servidor de desenvolvimento:

   npm run dev


## Estrutura do projeto

- src/
  - main.jsx — entrada da aplicação
  - App.jsx — componente principal
  - components/Counter/Counter.jsx — exemplo de componente com ciclo de vida

## Observações

- Os consoles do navegador mostram logs que ilustram o ciclo de vida dos componentes.
